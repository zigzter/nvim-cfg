# zigvim

After using a few different pre-built Neovim plugins (kickstart.nvim & LazyVim), I decided to try building my own from scratch. Forever a work in progress, more than likely.

- telescope
- vim-tmux-navigator
- mason
- nvim-cmp
- nvim-treesitter
- lualine
- noice
- catppuccin
- dashboard-nvim
- Comment.nvim
